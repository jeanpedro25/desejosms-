# DesejosMS - Site de Acompanhantes

Um site moderno e responsivo para classificados de acompanhantes em Mato Grosso do Sul, similar ao HotMS.

## 🚀 Características

- **Design Responsivo**: Funciona perfeitamente em desktop, tablet e mobile
- **Interface Moderna**: Design limpo e profissional
- **Navegação Intuitiva**: Categorias organizadas (Garotas, Travestis, Garotos)
- **Seletor de Localização**: Filtro por cidades de MS
- **Modal de Detalhes**: Informações completas dos perfis
- **Contatos Diretos**: Links para telefone e WhatsApp
- **Política de Privacidade**: Termos e condições claros
- **Sistema de Anúncios**: Painel completo para anunciantes criarem seus anúncios
- **Painel Administrativo**: Gerenciamento completo de anúncios e usuários
- **Sistema de Login**: Autenticação para anunciantes
- **Upload de Fotos**: Galeria de imagens para cada anúncio
- **Estatísticas**: Métricas de visualizações e contatos

## 🚀 Como Usar

### Para Visitantes
1. Abra o arquivo `index.html` em qualquer navegador
2. Navegue pelas categorias (Garotas, Travestis, Garotos)
3. Selecione uma localização no dropdown
4. Clique em qualquer card para ver detalhes
5. Use os botões de contato para ligar ou enviar WhatsApp

### Para Anunciantes
1. Clique no botão "Anunciar" na página principal
2. Crie uma conta ou faça login
3. Preencha as informações do seu anúncio
4. Adicione fotos e serviços oferecidos
5. Gerencie seu anúncio através do painel

### Para Administradores
1. Clique no botão "Admin" na página principal
2. Acesse o painel administrativo
3. Gerencie todos os anúncios e usuários
4. Use os filtros para encontrar anúncios específicos
5. Edite, ative/desative ou exclua anúncios
