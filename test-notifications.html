Teste do Sistema de Notificações

Como usar o sistema de notificações:

  • showSuccess(message) - Notificação de sucesso
  • showError(message) - Notificação de erro
  • showWarning(message) - Notificação de atenção
  • showInfo(message) - Notificação informativa
  • showConfirm(message) - Notificação de confirmação
  • notifications.showList(title, items) - Notificação com lista
  • notifications.showProgress(title, progress) - Notificação com progresso

Recursos:

  • Animações suaves de entrada e saída
  • Auto-fechamento após 3 segundos (configurável)
  • Design responsivo para mobile
  • Diferentes tipos de notificação com cores distintas
  • Botões customizáveis
  • Suporte a HTML nas mensagens
