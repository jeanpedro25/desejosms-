Teste do Sistema de Verificação

Como funciona o sistema de verificação:
  1. O anunciante envia documentos pelo Painel Anunciante
  2. Os documentos ficam em "pendingVerifications" no localStorage
  3. O admin acessa a página de Verificações
  4. O sistema mescla automaticamente os documentos pendentes
  5. O admin pode aprovar ou rejeitar os documentos
