🚀 DesejosMS - Servidor Local

📋 Como acessar o projeto:

1. Abra o terminal/PowerShell na pasta do projeto
2. Execute: node server-fixed.js
3. Ou execute: python -m http.server 3025
4. Acesse: http://localhost:3025

🔗 Links Diretos:

🏠 Página Principal 📄 Index.html 👤 Painel Anunciante ✅ Verificações ⚙️ Admin Panel

💡 Dicas:

• Se os links não funcionarem, verifique se o servidor está rodando

• Use Ctrl+C no terminal para parar o servidor

• O projeto está na pasta: Projeto/Teste/
